Программа предназначена для создания и заполнения эксель таблицы данными из htm страниц: 
	1. Все работы автора на сайте. 
	2. Описание работ автора. 
Так же необходимо иметь аккаунт со всеми правами(аккаунт с абон. платой).

ПОДГОТОВКА ФАЙЛОВ.

1.Логинимся на сайте Scopus.com
2.В строке "Поиск авторов" вводим фамилию автора.
3.Выбираем "нашего" автора. Переходим на страницу его работ.
4.ПКМ по странице "Сохранить как".(Это первый файл).
5.Переходим на страницу "Библиографическое описание" и так же сохраняем страницу.(Это второй файл).
	
ДЛЯ РАБОТЫ С ПРОГРАММОЙ НЕОБХОДИМО:
   1. создать отдельную папку
   2. перенести программу в эту папку.
   3. сохранить результаты поиска на сайте Scopus (Сохранить страницу как) в созданную вами папку с программой ВАЖНО!!!!! Сохраненная страница ОБЯЗАНА называться Scopus.htm (Как в прикрепленном файле).
		3.1 Updated: Для работы программы необходимо в папку положить файл с библ описанием (processRequesst.htm) Без него программа работать не будет.
   4.Запустить файл Start.bat
   5. При успешном запуске в папе появится файл Scopus.xls
  
  При повторном запуске программы (Start.bat) данные файла Scopus.xls будут затираться новыми данными.
  !!!!!! Каждая, каждая ... следующая сохраненная страница должна называться Scopus.htm

П.С. Использовал триал версию аккаунта. На сайте много функций заблокировано и скрыто для таких пользователей, поэтому для парсинга использовались сохраненные страницы с прем аккаунта... Программа создавалась для "Премиум" аккаунта.
